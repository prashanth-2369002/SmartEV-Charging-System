## Creation Tools

**Free tools to create the banner:**

1. **Canva** (canva.com) — drag-and-drop; use "GitHub Profile Banner" template
2. **Figma** (figma.com) — precise design; export at 1280×640 PNG
3. **Adobe Express** — free tier available; EV/tech templates available

**Icon resources:**
- Battery icon: Heroicons, Phosphor Icons
- Circuit board: Flaticon (free tier)
- EV/car icon: Font Awesome, Material Icons
